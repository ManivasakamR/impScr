Welcome to script importer for html/Javascript

Steps

Add a script tag inside head tag

<Script src="http://snipimp.herokuapp.com/loadScript.js"></script>

Call the function loadScript(url,no) inside the script tag of body for load script from stackoverflow answers

loadScript('https://stackoverflow.com/questions/7944239/generating-fibonacci-sequence',4);

Create a function for html click event. inside this function call the loaded function

getFibonacciNumbers(20);
